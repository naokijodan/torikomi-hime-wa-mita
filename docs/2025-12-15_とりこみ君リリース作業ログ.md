# とりこみ君 リリース作業ログ

**作業日**: 2025-12-15
**作業内容**: Universal Scraper → とりこみ君への名称変更、アイコン更新、Chrome ウェブストア公開準備

---

## 1. 背景・目的

### 1.1 名称変更の経緯
- 元の名称「Universal Product Scraper」は機能的だが覚えにくい
- 「Scraper」という単語がChrome審査でネガティブに見られる可能性
- より親しみやすく、日本語で覚えやすい名前を検討

### 1.2 名称候補の検討
| 候補 | 評価 |
|-----|------|
| トレカタ | 取れる+カタログ |
| ひろい屋 | 和風 |
| 商品ピッカー | 機能的 |
| とりこみ君 | シンプル、親しみやすい ✓ |
| ポチっととりこみ君 | やや長い |

**最終決定**: 「とりこみ君」（5文字、覚えやすい、機能が直感的に伝わる）

### 1.3 Chrome ウェブストアの文字数制限
| 項目 | 制限 |
|-----|------|
| name（名前） | 最大75文字 |
| short_name（短縮名） | 推奨12文字以内 |
| description（説明） | 最大132文字 |

「とりこみ君」は5文字なので問題なし。

## 2. アイコンデザイン

### 2.1 コンセプト検討
- 笑顔のボックス → 宝箱に商品が入っていくイメージ
- ゆるキャラ風のかわいい宝箱

### 2.2 最終デザイン
- 開いた蓋の宝箱
- 大きな目とにっこり笑顔
- ピンクのほっぺ
- 中にキラキラ（✨）
- パステル調の茶色とゴールド
- 背景透明

### 2.3 アイコン生成
- ツール: ChatGPT (DALL-E)
- サイズ: 1024x1024px → 128x128pxにリサイズ
- フォーマット: PNG（背景透明）

**使用したプロンプト**:
```
ゆるキャラ風のかわいい宝箱のアイコン。小さな手足がついている。
大きくてキラキラした目、ほっぺがピンク、にっこり笑顔。
蓋が開いていて、中にキラキラの光（✨）が見える。
商品は持たない。シンプルな線、丸みのあるデザイン、
パステル調の茶色とゴールド、背景透明、アプリアイコン用128x128px
```

## 6. Chrome ウェブストア提出手順

### 6.1 準備物
- ZIPファイル: `torikomi-kun-v1.2.1.zip`
- 場所: `/Users/naokijodan/Desktop/universal-scraper/`

### 6.2 提出手順
1. Chrome デベロッパーダッシュボードを開く
   - https://chrome.google.com/webstore/devconsole

2. 既存の拡張機能「Universal Product Scraper」を選択

3. 新しいバージョンをアップロード
   - 左メニュー「パッケージ」→「新しいパッケージをアップロード」
   - `torikomi-kun-v1.2.1.zip` をアップロード

4. ストア情報を更新
   - 左メニュー「ストア掲載情報」
   - 名前を「とりこみ君」に変更
   - 必要に応じて説明文も更新

5. 審査に提出
   - 右上の「審査のため送信」をクリック
   - 審査完了まで数時間〜数日
